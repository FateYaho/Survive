# 원랜디 세부 조합표

## 사용 목적

이 문서는 원랜디식 랜덤 조합 디펜스 게임을 설계할 때 참고하기 위한 세부 조합표다.

정리 범위: 안흔함 / 특별함 / 희귀함 / 전설적인 / 히든 / 초월 / 불멸 / 영원 / 제한

> **주의**: 이 문서는 현재까지 수집한 ORDSearch 화면/정리 내용을 바탕으로 만든 참고용 조합표다. 실제 최신 버전과 일부 차이가 있을 수 있다. 네 게임을 만들 때는 캐릭터명/IP를 그대로 쓰지 말고, 조합 구조와 난이도 패턴만 참고하는 용도로 사용한다.

## 12. 참고 메모

이 조합표에서 참고해야 할 것은 캐릭터명이 아니라 다음 구조다.

1. 초반은 2개 조합으로 쉽게 시작한다.
2. 중반부터는 3개 조합으로 목표감을 만든다.
3. 전설부터는 목재 같은 자원을 요구한다.
4. 초월은 공통 게이트 재료로 진입을 통제한다.
5. 불멸/영원은 최종 목표 역할을 한다.
6. 제한은 비용/조건으로 숙련도를 요구한다.

> **한 줄 요약**: 조합표는 단순 데이터가 아니라, 유저가 매 판 목표를 세우게 만드는 장치다.
