# 🎮 영토 개척 생존기 (가제)

> Dome Keeper + Slay the Spire 스타일의 서바이벌 로그라이크 게임

낮엔 영토를 넓히며 자원을 캐고, 밤엔 몰려오는 적으로부터 코어를 지켜라. 수집한 자원으로 특화 테크를 타며 매번 다른 빌드로 생존하는 로그라이크 서바이벌.

- 🤖 **[Claude 작업 컨텍스트 (CLAUDE.md)](CLAUDE.md)** - AI 작업 시 참조

## 🎯 핵심 특징

- ⚔️ **낮/밤 듀얼 루프**: 파밍 ↔ 전투 ↔ 사고의 리듬
- 🌳 **수평 특화 테크**: 나무/돌/철/마법 4계열 중 선택·조합
- 🗺️ **영토 확장 = 전략**: 확장은 공격적 + 방어적 이중 결정
- 🎲 **로그라이크 런**: 30~60분 한 판, 매번 다른 경험
- 🐉 **4부위 보스**: 3가지 클리어 경로 (특화/범용/융합)

---

## 🛠 기술 스택

- **Phaser 3** — 게임 엔진
- **TypeScript** — 언어
- **Vite** — 번들러

## 🚀 개발 시작

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 빌드
npm run build
```

> ⚠️ 아직 Vite 프로젝트 초기화 전입니다. 세팅 후 위 명령어로 실행 가능.
