## 0. 한 줄 요약

**랜덤 조합 디펜스 + 직장인 테마 + 루프 맵.**

- **주요 레퍼런스**: 원랜디 (원신 랜덤 디펜스), 운빨존망겜 [벤치마킹 자료 수령 후 반영 예정]

## 3. 맵 구조

- **루프형 경로**: 적이 원형 경로를 뱅글뱅글 돌며 반복 순환. 도착점 없음.
- **게임오버 조건**: 경로 위 동시 생존 적 수가 한도(X)를 초과하면 "직원 처리 한도 초과" → 게임오버
  - X 수치는 밸런싱 시 결정 [TBD]
- **유닛 배치**: 경로 주변 타일에 직원 유닛 배치

## 5. 등급 체계

낮은 등급 유닛을 조합해 상위 등급으로 압축하는 것이 핵심.

| 등급 | 직급 | 비고 |
|------|------|------|
| 1등급 | 인턴 | 기본 |
| 2등급 | 사원 | |
| 3등급 | 대리 | |
| 4등급 | 과장 | 스토리 보상 등급 |
| 5등급 (전설) | 팀장 / 부장 | |
| 히든 | C레벨 (CTO, CMO 등) | 외국계 기업 느낌, 크로스 조합 결과 |
| 불멸 | 사장 | |
| 영원 | 회장 | |
| 제한 | 금수저 | 과장부터 시작하는 특수 유닛 |

- **조합 방식**: 빌드 페이즈에서 같은 등급 유닛 2개 선택 → 배치 위치 선택 → 상위 등급 유닛으로 합체
- **금수저**: 과장 등급 유닛으로 시작, 원랜디 레일리 방식 (합성 가능)
