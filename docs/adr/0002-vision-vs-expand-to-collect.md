# ADR-0002: 시야 시스템 도입 + 채집 조건 완화 (Expand-to-Collect 폐기)

**날짜**: 2026-04-25
**상태**: 적용됨

## Context (배경)

원래 설계:
- 모든 비-OWNED 타일은 FOG (안 보임)
- 자원은 OWNED 타일에서만 채집 가능
- 따라서 자원을 얻으려면 먼저 확장(10W+5S 비용) 필요

문제:
- 사이클 1 시작 시 인벤토리 20W+10S → 확장 2번이면 자원 0
- 그 2번이 빈 타일이면 끝장. 다시 일어설 자원 없음 (**데드락**)
- 사용자 피드백: "맵 여는데 자원이 없으면 끝이야. 망해"
- "차라리 맵이 다 안 보이는 상태에서 가면 뭐가 있는지 보이고 거기서 자원을 먹을 수 있게"

## Decision (결정)

1. **시야 시스템 도입**:
   - `Player.visionRadiusTiles = 4` (체비셰프 거리)
   - 플레이어 이동 시 주변 4타일 반경의 FOG → EXPLORED 전환
   - 스폰 직후에도 1회 reveal (초기 9×9 가시 영역)

2. **채집 조건 완화**:
   - OWNED 전용 → **EXPLORED·OWNED 둘 다 가능**
   - FOG는 여전히 채집 불가 (안 보이니까)

3. **확장의 역할 재정의**:
   - 더 이상 "자원 채집 필수 조건"이 아님
   - 이제 "건물 배치를 위한 영토 확보" + "몬스터 어그로 영역 확장" 용도

## Consequences (영향)

- ✓ 데드락 해소: 시작 인벤토리만으로도 걸어다니며 자원 확보 가능
- ✓ "탐험적 즐거움" Core Pillar #5 강화
- ✓ 확장은 의미 있는 트레이드오프로 남음 (건물 짓고 싶으면 확장 필요)
- ✗ 게임 정체성 "영토 개척" 의미 일부 약화 (확장 안 해도 플레이 가능)
- ✗ TileMap render에서 FOG/EXPLORED 시각 차이 명확해야 함 (현재 OK, 색 분리 됨)
- 🔧 후속 결정: 확장 비용 계단식 스케일링 (ADR-0003) — 영토 의미 보강
