## 🧮 수입 계산 공식

### A. DAY 플레이어 채집

**기본 단위**
```
타일 크기:    32px
플레이어:     128 px/s = 4 타일/s
타일당 자원:  8개 (collectAmount 2 → 4회 채집으로 소진)
채집 시간:    나무 1s/회 × 4 = 4s/타일, 돌 1.5s/회 × 4 = 6s/타일
타일 간격:    minSpawnGap 2타일 → 평균 이동 0.75s
```

**타일당 총 소요**
```
나무: 4s 채집 + 0.75s 이동 = 4.75s → 8W
돌:   6s 채집 + 0.75s 이동 = 6.75s → 8S
혼합: 0.65 × 4.75 + 0.35 × 6.75 = 5.45s
```

**Day 1 (90s) 최대 채집 — 건물 즉시 배치, 복귀 이동 없음 가정**
```
초반 이동 (중앙 → 첫 자원): 1.5s
유효 채집 시간: 88.5s

혼합 채집: 88.5 / 5.45 ≈ 16타일
  나무 ~10타일 → 80W
  돌  ~ 6타일 → 48S
  + 시작 인벤토리 30W+15S
  = 최대 110W + 63S

나무만 채집:  88.5 / 4.75 ≈ 18타일 → 144W (+ 15S 시작)
```

**현실 보정 (경로 판단·탐색 손실 ~25%)**
```
Day 1~3 현실:   75~90W + 40~50S
Day 4~5 (150s): 120~150W + 65~85S
```

### B. NIGHT 드롭

```
공식: wolves × killRate × (2W + 1S)

| 사이클 | 100% 처치 | 70% 처치 |
|--------|----------|---------|
| 1 (10) | 20W+10S  | 14W+7S  |
| 2 (15) | 30W+15S  | 21W+11S |
| 3 (20) | 40W+20S  | 28W+14S |
| 4 (25) | 50W+25S  | 35W+18S |
| 5 (30) | 60W+30S  | 42W+21S |
```

### C. 생산 건물

```
제재소: DAY 90s / 5s = 18틱 × 2W = 36W/사이클 (early)
        DAY 150s / 5s = 30틱 × 2W = 60W/사이클 (mid)
채석장: DAY 90s / 8s = 11틱 × 2S = 22S/사이클 (early)
        DAY 150s / 8s = 18틱 × 2S = 36S/사이클 (mid)
```

### D. 방어 DPS

```
아군 총 DPS = (터렛 수 + 1) × 10
총 적 HP = wolves × 20
이론 처치 시간 = 총 적 HP / 총 아군 DPS

현실 보정: 실전 필요 터렛은 이론의 2~4배
  → 목표 터렛 수: 사이클 1=1~2 / 2=2~3 / 3=3~4 / 4=5~6 / 5=7~9
```
