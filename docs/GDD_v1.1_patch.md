- **코어 파괴가 유일한 게임 오버 조건** (플레이어 사망 ≠ 게임 오버)

## §5.4 가공 자원 (전면 재설계)

**Before 제목**: "희귀 자원" — 고목, 생명의 나무, 세계수 등을 "자원"처럼 나열

**After 제목**: **"가공 자원 (Phase 2+)"**

**After 내용**:

각 계열의 특화 건물은 건설된 뒤 **매 밤 종료 시 가공 자원 1개 생산**. T3~T4 테크 연구의 필수 재료.

| 생산 건물 | 가공 자원 | 용도 |
|----------|----------|------|
| 🌳 정령의 숲 | **정령 수액** | 나무 T3~T4 연구 |
| ⚒️ 대장간 | **룬 파편** | 돌 T3~T4 연구 |
| 🏭 공장 | **기계 부품** | 철 T3~T4 연구 |
| 🔮 마법 연구소 | **마나 결정** | 마법 T3~T4 연구 |

**규칙**:
- 밤 동안 건물 파괴 시 해당 밤 생산분 없음 → **건물 방어 동기 강화**
- 생산량은 건물당 사이클당 1개 (초안). Phase 2 플레이 테스트로 튜닝.
- ⚠️ Phase 1 MVP에서는 가공 자원 시스템 **미구현**. 정령의 숲은 시설 효과(+20% 채집) + T2 해금 gate만.

**명칭 혼란 정리**:
> "고목", "생명의 나무", "세계수" 같은 이름은 **테크 단계의 명칭**이지 자원이 아님. 해당 테크가 해금되면 플레이어/터렛 효과가 바뀌는 것이지, "고목"이라는 아이템이 인벤토리에 쌓이지 않음. 자세한 테크 단계는 §6.3 참조.

## §6 테크 & 건물 시스템 (**대규모 재설계**)

### §6.1 핵심 원칙 (전면 교체)

**Before**:
- 즉시 적용 + 건물 기반 구조
- T1은 연구실, T2+는 계열별 특화 건물 필요
- 건물 자체가 시설 효과 (이중 보상)

**After**:
- **T0~T4 5단계 체계** (T0=원재료, T4=정점)
- **원재료만으로 T1~T2 연구 / 원재료 + 가공 자원 조합으로 T3~T4 연구**
- **3범주 연구 옵션** (무기 / 스탯·장비 / 방어 시설) 중 자원 부족으로 부분 선택 → 빌드 다양성의 근원
- 건물 자체가 시설 효과 + T3~T4 전용 가공 자원 생산
- 공간·자원 제약이 전략성 확보 → 턴제 불필요

### §6.2 테크 진행 흐름 (재설계)

**After**:

```
T0 (원재료)
  └─ 자원 타일에서 채집 (나무/돌/철/금)

T1 (기본 가공)
  └─ 연구실에서 원재료만 소비 → 즉시 적용
  예: 단단한 나무, 돌도끼, 나무 벽, 기본 터렛

T2 (특화 진입)
  └─ 해당 계열 특화 건물 건설 (정령의 숲/대장간/공장/마법 연구소)
  └─ 건물 자체가 시설 효과 + 연구 gate

T3~T4 (중급·정점)
  └─ 특화 건물에서 연구: 원재료 + 해당 건물이 밤마다 생산한 가공 자원
  예: 나무 T3(생명의 나무) = 나무 300 + 정령 수액 3
```

### §6.2 추가: 3범주 연구 옵션

각 티어에서 복수 연구 옵션 제공. 한 런에 자원 부족으로 전부 불가능 → **빌드 다양성**.

| 범주 | 내용 |
|------|------|
| 1. 플레이어 무기 | 주력 무기 업글 / 신무기 해금 |
| 2. 플레이어 스탯·장비 | HP·공격력·리젠·방어력 영구 보너스 |
| 3. 방어 시설 | 터렛 / 벽 / 광역 시설 (T3+는 **특수 스킬** 부여) |

### §6.3 4계열 테크 트리 (유닛 → 터렛 변환)

**Before** (나무 계열 예시):

```
나무 → 단단한 나무 → 고목 → 생명의 나무 → 세계수
                         ↓
                      정령 나무 (자동 공격 유닛)
```

**After**:

```
T0 나무 → T1 단단한 나무 → T2 고목 → T3 생명의 나무 → T4 세계수
                                ↓
                        정령 터렛 (자동 공격 + 아군 회복)
```

"자동 공격 유닛" → **"자동 공격 터렛"** 용어 통일. 유닛 단어 전면 삭제.

**돌 계열**: "룬스톤 요새 (광역 실드)" — 변경 없음 (원래 시설)

**철 계열**: "자동 공장 (자원 자동 생산)" — 변경 없음. Phase 2+ 가공 자원(기계 부품) 생산 시스템과 일치

**금·수정 계열**:
- "수정"이 자원에서 삭제됐으므로 **계열명: "금 계열 — 마법/아케인"** 으로 변경
- 트리: `T0 금 → T1 정제된 금 → T2 마력 결정 → T3 신성 결정 → T4 창조의 보석`
- "차원 결계 (무적 시간)" — 변경 없음

- 매 프레임 재탐색 X → 구현 단순 + 플레이어의 "미끼 유도" 전략 가능

**2. 히어로 → 몬스터 (자동 공격)**

- 매 공격마다 **가장 가까운 몬스터** 재탐색
- 수동 조준 불필요 → 플레이어는 이동·회피에만 집중

**3. 방어 터렛 → 몬스터**
- 사거리에 처음 진입한 몬스터 **타깃 고정**
- 타깃 사망 또는 사거리 이탈 시 재탐색
- **몬스터와 동일한 규칙**: 구현 일관성
