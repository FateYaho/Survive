## 🏗️ 테크 단계 체계 (신규)

| 단계 | 정의 | 해금 조건 |
|------|------|----------|
| **T0** | 원재료 (자원 자체) | 채집 즉시 |
| **T1** | 기본 가공 | 연구실에서 원재료 소비 → 즉시 적용 |
| **T2** | 특화 진입 | 특화 건물 건설 (정령의 숲/대장간/공장/마법연구소) |
| **T3** | 중급 특화 | 원재료 + 해당 건물 **가공 자원** |
| **T4** | 정점 | 원재료 大 + 가공 자원 多 |

### 가공 자원 (Phase 2+)

각 특화 건물이 **매 밤 종료 시 1개 생산**. 건물 파괴 시 해당 밤 생산분 없음.

| 생산 건물 | 가공 자원 | 용도 |
|----------|----------|------|
| 🌳 정령의 숲 | 정령 수액 | 나무 T3~T4 |
| ⚒️ 대장간 | 룬 파편 | 돌 T3~T4 |
| 🏭 공장 | 기계 부품 | 철 T3~T4 |
| 🔮 마법 연구소 | 마나 결정 | 마법 T3~T4 |

> ⚠️ Phase 1 MVP에서는 가공 자원 시스템 미구현. 정령의 숲은 시설 효과(+20% 채집) + T2 해금 gate만.

## 🎁 T3/T4 보상 구조

**3범주 연구 옵션** — 한 런에 자원 부족으로 다 못 찍음. 이게 빌드 다양성의 근원.

1. **플레이어 무기** — 주력 무기 업글 (예: 정령의 활 → 세계수의 활)
2. **플레이어 스탯/장비** — HP·공격력·리젠 보너스
3. **방어 시설 (터렛·벽·광역 시설)** — 새 터렛 + **특수 스킬**

### 터렛 특수 스킬 예시 (T3/T4)

| 계열 | T3 | T4 |
|------|-----|-----|
| 🪵 나무 | 정령 터렛 (공격+아군 회복) | 세계수 코어 (맵 전역 자연 리젠) |
| 🪨 돌 | 룬 포병 (광역+스턴) | 고대 성체 (광역 실드) |
| ⚙️ 철 | 레일건 (고DPS 관통) | 자동 공장 터렛 (방어+자원 자동 생산) |
| ✨ 마법 | 원소 포탑 (지속 화염/빙결) | 차원 결계 (특정 영역 무적) |

### 유닛 소환 시스템 → **삭제**

- 이유: 구현 난이도(경로 탐색·AI·타깃팅) 대비 재미 기여 낮음
- 모든 "유닛"성 요소를 **터렛·시설**로 통합
- Phase 5 엔드 컨텐츠 수준에서 재검토 여지 남김

### 융합 테크 → **v1.1에서 전면 제외** (2026-04-24 추가)

- 원래 v1.0에 있던 4개 융합 조합(엘프 마법사 / 기계 골렘 / 레일건 / 드래곤 소환) **모두 삭제**
- 이유: 각 계열 T0~T4 독립 트리를 먼저 깊이 파는 데 집중. 조합 복잡도 vs 재미 기여가 불확실
- 4×4 구조(4계열 × T1~T4)로 단순화 → 빌드 다양성은 **3범주 연구 옵션**(무기/스탯/시설)에서 확보
- **Phase 5+ 확장 컨텐츠**로 재도입 검토 (OPEN_ISSUES P3 참조)

### 몬스터 "박쥐" 대체 → **"그림자"** 확정 (2026-04-24 추가)

- 공중 박쥐 삭제 → **"👤 그림자(Shadow) — 지상 추적형"** 으로 대체
- 다크 톤 통일성 확보(유령과 계열 맞음), 추적형 이동 패턴 표현
